The HTML file implements basic functionality of configuring a network device.
The page has been developed using basic HTML, jquery, javascript and css.
Open the device-config.html file to view the webpage.
On right clicking the image a menu is displayed listing the various configuration options. On clicking on any one of the options a dialog box is displayed seeking input from the user. Basic input validation for IP addresses has been implemented. On clicking the applied button an alert is displayed signifying that the changes have been applied. On clicking the cancel button the dialog box ceases to be displayed.
